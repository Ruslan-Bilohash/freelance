=== Freelance CMS ===

Universal PHP freelance marketplace / job board script for Norway, Europe and Ukraine.
Upwork/Fiverr-style demo: projects, proposals, freelancer profiles, Simple/Pro tiers, admin panel.

Languages: Norwegian (default), English, Ukrainian, Russian — 100% UI + admin

Features
  - 12 demo projects + 6 freelancer profiles
  - Clickable skill tags (Elementor, WordPress, Copywriting → search)
  - Categories dropdown in header (mobile-friendly)
  - Become-a-freelancer landing + 9 SEO city/region pages
  - Admin: projects, proposals, freelancers, users, settings
  - Settings: site/button colours, background, reCAPTCHA, Grok/GPT chat
  - Schema.org, hreflang, sitemap, emerald green responsive UI

Tech
  PHP 8+, JSON storage, no framework, Font Awesome 6

Setup
  1. Upload folder to /freelance/ — make data/ writable
  2. Admin → Settings → colours, reCAPTCHA, AI chat
  3. Edit lang/*.php for extra locales
